## 📖 项目简介

**GeoClaw** 是专为地质科研工作者设计的 AI 智能助手，将大语言模型与地质专业知识深度融合，实现：

- 🔍 **文献智能检索** - 基于语义理解的精准文献定位
- 🖼️ **多模态图像分析** - 岩芯、薄片、遥感影像智能识别
- 📝 **自动化报告生成** - 符合学术规范的地质调查报告
- 📊 **数据分析可视化** - 地质数据的智能分析与展示

## 🎯 核心价值

| 功能 | 传统方式 | GeoClaw 方式 | 效率提升 |
|------|----------|-------------|---------|

## 🚀 快速开始

### 1. 安装依赖

```bash
cd ~/openclaw-game
pip install -r requirements.txt
```

### 2. 配置 API Key

```bash
# 创建配置文件
mkdir -p ~/.openclaw
echo "ZHIPU_API_KEY=your_api_key_here" > ~/.openclaw/.env
```

获取智谱 API Key: https://bigmodel.cn/usercenter/proj-mgmt/apikeys

### 3. 启动应用

```bash
cd ~/openclaw-game/app
streamlit run geoclaw.py
```
