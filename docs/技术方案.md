# GeoClaw 技术方案

## 一、项目背景

### 1.1 赛事背景
中关村北纬龙虾大赛是由北京中关村学院、中关村人工智能研究院主办的 AI 应用创新赛事，旨在推动 AI Agent 技术在垂直领域的落地应用。

### 1.2 参赛赛道
**学术龙虾赛道**：所有跟"搞懂一件事"有关的场景，都是它的主场。

### 1.3 项目定位
GeoClaw 面向地质科研工作者，通过 AI 技术提升文献检索、图像分析、报告生成等工作的效率。

## 二、技术方案

### 2.1 核心模型选择

| 模型 | 用途 | 选择理由 |
|------|------|---------|
| GLM-4-Flash | 对话理解与推理 | 国产领先模型，中文能力强，性价比高 |
| GLM-4V-Flash | 视觉理解与图像分析 | 支持地质图像识别，多模态能力强 |
| 智谱 ASR | 语音转文本 | 支持野外语音记录转写 |

### 2.2 技术架构

```
┌─────────────────────────────────────────┐
│           用户交互层 (Streamlit)         │
├─────────────────────────────────────────┤
│           业务逻辑层 (Python)            │
│  文献检索 │ 图像分析 │ 报告生成 │ 数据分析 │
├─────────────────────────────────────────┤
│           技能扩展层 (OpenClaw Skills)   │
│  文件系统 │ 网页抓取 │ 视觉分析 │ 语音转写 │
├─────────────────────────────────────────┤
│           模型服务层 (智谱 AI)           │
│    GLM-4-Flash  │  GLM-4V-Flash  │ ASR  │
└─────────────────────────────────────────┘
```

### 2.3 核心功能实现

#### 文献智能检索
- 输入：用户研究需求描述
- 处理：GLM-4 语义理解 → 生成检索词 → 模拟检索
- 输出：相关文献推荐、数据来源建议

#### 岩芯图像分析
- 输入：岩芯/薄片/遥感图像
- 处理：GLM-4V 视觉理解 → 矿物识别 → 结构分析
- 输出：分析报告（岩性、矿物、结构、蚀变、找矿意义）

#### 自动化报告生成
- 输入：项目信息、工作内容、分析数据
- 处理：GLM-4 文本生成 → 结构化组织 → 规范化输出
- 输出：符合地质行业规范的调查报告

## 三、创新点

### 3.1 方法论迁移
借鉴 Nature 论文《AI agents for rare disease diagnosis》的方法论，将医疗领域的成功经验迁移至地质领域。

### 3.2 多模态融合
整合文本、图像、语音多种模态，实现地质信息的全方位理解与分析。

### 3.3 自动化工作流
从数据采集到报告生成，提供一站式解决方案，显著提升工作效率。

### 3.4 低代码使用
通过 Streamlit 构建 Web 界面，降低技术门槛，让地质工作者无需编程即可使用。

## 四、应用价值

### 4.1 效率提升
- 文献检索效率提升 3-5 倍
- 图像分析时间节省 80%
- 报告撰写效率提升 5-10 倍

### 4.2 质量保障
- 自动生成的报告符合地质行业规范
- AI 分析结果可作为人工判断的辅助参考

### 4.3 成本降低
- 减少重复性工作时间
- 降低对专业软件的依赖
